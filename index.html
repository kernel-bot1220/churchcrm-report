🗄️ 核心数据表

最核心的7张业务表及其关键字段，点击展开查看完整字段

人员主表 person_per
人员主数据，包含姓名、地址、电话、邮箱等核心信息。所有人员的唯一标识，per_fam_ID 关联家庭，per_cls_ID 关联分类，per_fmr_ID 关联家庭角色。per_FriendDate 标记首次到访访客。
字段名类型说明
家庭主表 family_fam
家庭主数据，包含家庭名、地址、电话、邮箱。教会以家庭为单元进行奉献统计和通讯管理。fam_Latitude/fam_Longitude 支持地图展示。
字段名类型说明
小组主表 group_grp
小组定义，包含小组名、类型（grp_Type）、角色列表（grp_RoleListID）。小组是教会人员组织的基本单元，通过 person2group2role_p2g2r 建立人员与小组的多对多关系。
字段名类型说明
活动主表 events_event
活动定义，包含活动名、类型、时间、地点、联系人。通过 event_attend 记录参与人，通过 event_audience 定义目标受众小组，通过 kioskassginment_kasm 关联签到终端。
字段名类型说明
奉献记录表 pledge_plg
奉献承诺与支付记录，plg_PledgeOrPayment 区分承诺(Pledge)和支付(Payment)。关联家庭（plg_FamID）、资金科目（plg_fundID）、存款批次（plg_depID）。支持多种支付方式和定期计划。
字段名类型说明
存款批次表 deposit_dep
银行对账批次，包含存款日期、备注、关闭状态(dep_Closed)。一个批次可包含多笔 pledge_plg，通过 dep_Type 区分支票/现金等类型。
字段名类型说明
用户账号表 user_usr
系统用户，绑定到 person_per（usr_per_ID）。包含用户名、密码哈希、API Key、2FA密钥。权限字段包括 AddRecords、EditRecords、DeleteRecords、Finance、Admin 等。
字段名类型说明

📝 人员注册流程

// 1. 创建人员记录
$person = new Person();
$person->setFirstName($firstName);
$person->setLastName($lastName);
$person->setFamId($familyId);  // FK → family_fam
$person->setClsId($classification);  // 成员分类
$person->save();

// 2. 分配小组角色
$groupService->addUserToGroup($personId, $groupId, $roleId);
// → person2group2role_p2g2r 表插入记录

// 3. 活动签到
$eventAttend = new EventAttend();
$eventAttend->setEventId($eventId);
$eventAttend->setPersonId($personId);
$eventAttend->setCheckinDate(new DateTime());
// → event_attend 表记录

💰 奉献支付流程

// 1. 验证家庭
$family = FamilyQuery::create()->findOneById($famId);

// 2. 创建奉献记录 (Pledge or Payment)
$pledge = new Pledge();
$pledge->setFamId($famId);    // FK → family_fam
$pledge->setFundId($fundId);   // FK → donationfund_fun
$pledge->setAmount($amount);
$pledge->setSchedule($schedule);  // Weekly/Monthly/OneTime
$pledge->setPledgeOrPayment('Payment');  // 区分承诺和实际支付
$pledge->save();

// 3. 加入存款批次
$deposit = DepositQuery::create()->findOneById($depositId);
$deposit->addPledge($pledge);
$pledge->setDepId($depositId);  // FK → deposit_dep
$pledge->save();

// 4. 生成报税数据
$taxData = $financialService->getTaxReportData($fyId, $famId);

🔐 用户认证流程

// 1. 登录验证
$user = UserQuery::create()->findOneByUserName($username);

    // 2. 生成 Token → tokens 表
    $token = bin2hex(random_bytes(32));

    $tokenRecord->setType('api');
    $tokenRecord->setValidUntilDate($expiry);
    $tokenRecord->setReferenceId($user->getPerId());
